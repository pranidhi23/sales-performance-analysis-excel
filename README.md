# Sales Performance Analysis using Excel

## Project Overview
This project analyzes monthly sales data to evaluate revenue, cost, profit, and profit margins.
The objective is to identify trends, inefficiencies, and provide actionable business recommendations.

## Tools Used
- Microsoft Excel
- Formulas (Profit, Profit %)
- Bar Chart and Line Chart

## Key Metrics Analyzed
- Revenue
- Cost
- Profit
- Profit Margin (%)

## Key Insights
- April recorded the highest revenue but had the lowest profit margin due to increased costs.
- May demonstrated the highest efficiency with an 88% profit margin.
- High revenue does not always translate into high profitability.

## Business Recommendations
- Optimize costs during high-revenue months to improve profit margins.
- Monitor cost-to-revenue ratios regularly.
- Focus on efficiency, not just revenue growth.

## Outcome
This project demonstrates hands-on experience in business data analysis and visualization using Excel.
